# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-08

### Added
- **dotnet-cqrs-api**: Building .NET APIs using CQRS pattern with MediatR, FluentValidation, and Carter
- **maker**: Generate .NET Core data API code from model definitions using Maker XML specification
- **reindex-docs**: Re-index PDF and HTML documents and maintain static document repositories
- **repo-docs**: Update and maintain core repository documentation files (README, CHANGELOG, LICENSE, CONTRIBUTING)

## [1.0.0] - 2026-01-08

### Added
- Initial aspose-slides skill for PowerPoint presentation manipulation
  - Comprehensive API reference documentation
  - Implementation guide with practical examples
  - Resource organization guide
  - Quick reference for common operations
  - Complete Aspose.Slides XML documentation
