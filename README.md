### dotnet-cqrs-api
Building .NET APIs using CQRS pattern with MediatR, FluentValidation, and Carter modules.

### maker
Generate .NET Core data API code from model definitions using Maker XML specification.

### reindex-docs
Re-index PDF and HTML documents and maintain static document repositories.

### repo-docs
Update and maintain core repository documentation files (README, CHANGELOG, LICENSE, CONTRIBUTING).
